### Video Manager (v0.0.1)

FontEnd tech stack used
- React
- React-route-dom


Installation steps
- Clone this repository
- #npm install
- #npm start
- Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

Run test
- #npm test

Run test coverage
- #npm test -- --coverage

Production Build
- #npm run build
- Production build is created in `build` folder and ready to deploy
